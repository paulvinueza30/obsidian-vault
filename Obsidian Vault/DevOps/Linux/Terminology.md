Linux kernel - glue between hardware and applications
Distro - Collection of programs combined with the linux kernel making up a Linux-based OS
Boot loader - program that boots the OS i.e. (GRUB)
Service - Program that runs as a background process
Filesystem: method for storing and organizing files
X Window System: Graphical subsystem on nearly all Linux systems
Desktop Environment: GUI on top of the OS
Command line: Interface for typing commands on top of the operating system
Shell: Command line interpreter that interprets the command line input and instructs the operating system to perform any necessary tasks and commands.
Filesystem Hierarchy Standard: standard form of how files are structured in every distro of Linux enforced by the Linux Foundation
Mounting: proccess of making a fs accessible at a certain point in the directory tree.
