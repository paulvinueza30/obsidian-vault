## SSH

Linux servers with process called sshd running

Use ssh client on Linux machines to use ssh format

ssh username@ipaddress -> were you may be prompted on a password depending if admin set that up

cloud environments have a specific location were you can disable/enable password login thru ssh

## Notes

1. passwords created for users are stored in /etc/shadow and they are one-way encrypted meaning that even if you have access to this file you cannot decrypt the password back
